# Apple Vinegar Order Form (سرکەی سێو)

This is a responsive, modern order form for Apple Vinegar, built with HTML, CSS, and Vanilla JavaScript.

## Features
- **Modern UI**: Glassmorphism design, RTL support (Kurdish Sorani), and smooth animations.
- **WhatsApp Integration**: Sends order details directly to WhatsApp (+9647515056972) with a pre-formatted message.
- **Responsive**: Works perfectly on mobile and desktop.

## How to Test Locally
1. Simply double-click `index.html` to open it in your browser.
2. Fill out the form and click "Send to WhatsApp".

## How to Deploy to Netlify

### Option 1: Drag & Drop (Easiest)
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag the `srkay sew` folder onto the page.
3. Your site will be live instantly!

### Option 2: Connect to GitHub (Recommended for updates)
1. Push this code to a new GitHub repository.
2. Log in to Netlify.
3. Click "Add new site" > "Import from existing project".
4. Select GitHub and choose your repository.
5. Click "Deploy site".

No build command is needed as this is a static site.
